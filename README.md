# Excel Sales Dashboard

## Project Overview
This project is an Excel Sales Dashboard created to analyze sales data and generate business insights.

The dashboard helps understand sales performance by region, category, customer segment, and monthly trends.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization

## Project Analysis
The dashboard includes analysis of:

- Total Sales by Region
- Total Sales by Category
- Total Sales by Segment
- Monthly Sales Trends
- Business Performance Metrics

## Dashboard Features
- Interactive charts
- Data filtering using slicers
- Sales trend visualization
- Easy-to-understand business reports

## Skills Demonstrated
- Data Cleaning
- Data Analysis
- Excel Dashboard Development
- Data Visualization
- Reporting
